# **University Management System**

A centralized database management system developed for the **Introduction to Database (CSC2108)** course at American International University-Bangladesh.

The project digitizes traditional university record management by providing an efficient relational database system for managing:

- Students
- Faculty Members
- Employees
- Courses
- Departments
- Semesters

---

# **Key Features**

## **Database Design**
- Designed using a complete **Entity Relationship (ER) Diagram**
- Models relationships between:
  - Students
  - Faculty
  - Employees
  - Departments
  - Courses
  - Semesters

---

## **Database Normalization**
- Normalized up to **Third Normal Form (3NF)**
- Reduces data redundancy
- Ensures data integrity and consistency

---

## **Relational Schema Implementation**
Implemented SQL schemas for:
- Employee
- Faculty
- Department
- Course
- Student
- Semester

---

## **SQL Operations & Querying**
Includes:
- Simple Joins
- Complex Joins
- Equijoins
- Self Joins
- Subqueries
- Aggregate Functions
- Database Views

---

## **Reporting & Analytics**
- Student CGPA analysis
- Structured data retrieval
- Optimized reporting queries
